# Verification Pass

## Purpose
Perform a final confidence pass and surface uncertainty honestly.

## When to use
- Work is close to completion
- You need an explicit confidence statement

## When not to use
- Early planning or architecture exploration

## Inputs
- Changed artifacts
- Checks performed
- Known uncertainties

## Outputs
- Verification summary
- Remaining uncertainty
- Confidence level

## Non-goals
- Claiming unperformed checks were completed
- Replacing project-specific CI
