# Prompt Runner Runbook

## Folder convention
artifacts/
  runs/
    YYYY-MM-DD/
      01_lead.md
      02_architecture.md
      03_implementation.md
      04_review.md

## Team habit
- Keep one run folder per significant change.
- Link the run folder in the PR description.
- Treat review output as part of the change record.

## Why this matters
This makes agent-assisted work inspectable and reproducible.
