# หลักสูตร

หลักสูตรนี้จัด feature full-stack ให้เป็นเฟสที่ประกอบกันได้

## เฟส 1: พื้นฐาน frontend

- โครงสร้างหน้า
- รูปแบบ state และ form
- loading และ empty state

## เฟส 2: การลงมือทำฝั่ง frontend

- การวาง App Router
- server/client boundary
- route handler เมื่อจำเป็น

## เฟส 3: พื้นฐาน backend

- API contract
- validation และ error handling
- auth และ permission shape

## เฟส 4: การลงมือทำฝั่ง backend

- โครงสร้าง service
- adapters และ repositories
- background jobs และ worker flow

## เฟส 5: การประกอบ end-to-end

- auth flow
- CRUD flow
- file flow
- async job flow

## สิ่งที่ต้องรู้ก่อน

- พื้นฐาน frontend ก่อน Next.js specifics
- พื้นฐาน backend ก่อน Python service specifics
- ทำ contract ก่อนลงมือเขียน implementation
