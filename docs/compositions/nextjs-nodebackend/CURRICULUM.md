# Curriculum

This curriculum reorganizes common full-stack features into modular phases.

## Phase 1: frontend fundamentals
- page structure
- state and form shape
- loading and empty states

## Phase 2: frontend implementation
- App Router placement
- server and client boundaries
- route handlers where needed

## Phase 3: backend fundamentals
- API contracts
- validation and error handling
- auth and permission shape

## Phase 4: backend implementation
- Node service structure
- handlers, services, repositories, adapters
- background jobs and integration points

## Phase 5: end-to-end composition
- auth flow
- CRUD flow
- file flow
- webhook/job flow

## Prerequisites
- frontend fundamentals before Next.js specifics
- backend common concepts before Node specifics
- contracts before implementation
