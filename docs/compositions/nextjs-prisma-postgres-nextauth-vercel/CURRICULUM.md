# Curriculum

This curriculum reorganizes the stack into modular phases.

## Phase 1: frontend fundamentals
- page structure
- state and form shape
- loading and empty states

## Phase 2: Next.js implementation
- App Router placement
- server and client boundaries
- server actions and route handlers

## Phase 3: backend fundamentals
- API contracts
- validation and error handling
- auth and permission shape

## Phase 4: data and auth implementation
- Prisma schema design
- migrations and seed strategy
- NextAuth.js configuration and session handling
- repository and adapter wiring

## Phase 5: deployment and review
- Vercel environment variables
- build and runtime constraints
- auth flow
- CRUD flow
- deployment verification

## Prerequisites
- frontend fundamentals before Next.js specifics
- backend common concepts before runtime specifics
- contracts before implementation
- schema and auth decisions before the first server action
