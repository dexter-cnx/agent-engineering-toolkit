# หลักสูตร

หลักสูตรนี้จัดสแต็กออกเป็นเฟสย่อย

## Phase 1: พื้นฐาน frontend
- page structure
- state และ form shape
- loading และ empty states

## Phase 2: การทำงานใน Next.js
- การวาง App Router
- server/client boundaries
- server actions และ route handlers

## Phase 3: พื้นฐาน backend
- API contracts
- validation และ error handling
- auth และ permission shape

## Phase 4: data และ auth implementation
- การออกแบบ Prisma schema
- migrations และ seed strategy
- การตั้งค่า NextAuth.js และ session handling
- การเชื่อม repository และ adapter

## Phase 5: deployment และ review
- Vercel environment variables
- build และ runtime constraints
- auth flow
- CRUD flow
- deployment verification

## สิ่งที่ควรมีมาก่อน
- frontend fundamentals ก่อน Next.js specifics
- backend common concepts ก่อน runtime specifics
- contracts ก่อน implementation
- schema และ auth decisions ก่อน server action แรก
