# Agent Engineering OS v0.1.0

## What this repository is

A stack-neutral Agent Engineering OS: canonical onboarding, runtime contracts, overlays-as-specialization, and CI-enforced governance checks.

## What this repository is NOT

- not a single-stack starter template
- not a mobile-only toolkit
- not an ungoverned prompt collection

## Supported surfaces

- Canonical: `docs/`, `agents/`, `memory/`, `system/`, `prompts/`, `tools/`, `overlays/`
- Compatibility: legacy root redirect docs (`START_HERE.md`, `HOW_TO_USE.md`, etc.)
- Legacy/reference: explicitly classified in `docs/reference/repo-surface-status.md`

## Known limitations

- Some legacy/reference surfaces remain for historical inbound compatibility.
- Overlay maturity differs (experimental/stable/production) and should be selected accordingly.
- Quickstart validates governance runtime, not end-to-end product deployment.
