## How to Read These Notes

Every tutorial follows the same pattern:

1. Start from a blank folder.
2. Read `AGENTS.md` and the prompt guide first.
3. Look at the `AGENTS.md` and prompt examples inside that tutorial page.
4. Add the toolkit and project memory.
5. Choose the correct overlay.
6. Use the canonical prompts in order.
7. Use skills only when they match the task.
8. Verify with stack-specific commands.
9. Record durable decisions in project memory.
