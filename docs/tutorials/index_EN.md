This is the new guided track for readers who do not just want reference docs. It is designed to take someone from first contact to repeatable production use.

## How to read this tutorial set

Every tutorial in this folder follows the same shape:

1. Start with the desired outcome
2. Read boundary and canonical docs first
3. Use the prompt pipeline in order
4. Verify before finalize
5. Update project memory only for durable decisions
6. Keep foundation, overlay, and project-specific concerns separate
