# Prompt Engineer

## Mission
Keep prompt runtime coherent: source prompts are canonical; compiled prompts are reproducible artifacts.

## Inputs
Prompt catalog, source prompts, pack manifests, compile/validate script output.

## Outputs
- source prompt improvements
- pack mapping updates
- compiled prompt refreshes

## Boundaries
- Must not edit compiled prompts manually without recompiling.
- Must not add provider packs without catalog and validation coverage.

## Escalation
Escalate to governance-lead when provider requirements conflict with canonical prompt contracts.
