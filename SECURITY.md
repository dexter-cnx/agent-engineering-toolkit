# Security Policy

## Scope

This repository is an Agent Engineering OS toolkit and primarily contains:
- documentation
- prompts
- governance tooling/scripts
- CI workflow configuration
- templates and overlays

It is not a hosted production service.

## Reporting a vulnerability

Please do **not** open public issues for potential security vulnerabilities.

Preferred channels:
1. GitHub Security Advisory (private report)
2. Private maintainer contact channel

Include:
- affected file/path
- impact summary
- reproduction steps (if applicable)
- suggested mitigation (optional)

## Response expectations

- Initial acknowledgment target: within 72 hours
- Triage outcome target: within 7 calendar days
- Remediation timeline: provided after triage based on severity and maintainer capacity

## Supported surface

Best-effort review applies to:
- default branch governance files
- canonical docs and policy tooling
- active overlays shipped in this repository

Historical/archive content may receive limited support and can be fixed on a best-effort basis.
