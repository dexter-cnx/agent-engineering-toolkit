# SKILL: flutter-notifications-fcm-production

## Purpose
Integrate Firebase Cloud Messaging in a production-safe way.

## Requirements
- Foreground, background, and terminated-state handling
- Notification permission flow isolated from UI
- Topic/subscription logic abstracted
- Navigation from notification payload handled centrally

## Deliverables
- Notification service abstraction
- FCM adapter
- Permission use case
- Payload parser
- Deep-link/navigation dispatcher
- Test coverage for parser and dispatcher
