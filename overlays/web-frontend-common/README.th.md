# โอเวอร์เลย์เว็บฟรอนต์เอนด์แบบทั่วไป

ใช้อโอเวอร์เลย์นี้สำหรับงานฟรอนต์เอนด์ที่เป็นกลางต่อเฟรมเวิร์ก: สถาปัตยกรรม UI, การไหลของ state และข้อมูล, ฟอร์ม, หน้า list/detail, loading/error/empty state และแนวทางทดสอบฝั่ง frontend

## ใช้ร่วมกับ
- `overlays/web-frontend-nextjs/` สำหรับรายละเอียด Next.js
- `docs/compositions/README.md` สำหรับตัวเลือกการประกอบ full-stack
- `docs/compositions/nextjs-dotnet/` สำหรับ Next.js + ASP.NET Core
- `docs/compositions/nextjs-python-service/` สำหรับ Next.js + Python service
- `docs/compositions/nextjs-nodebackend/` สำหรับ Next.js + Node backend
