# วิธีใช้งานโอเวอร์เลย์นี้

## มันคืออะไร
โอเวอร์เลย์นี้ให้ pattern ฟรอนต์เอนด์ที่ใช้ได้กับหลายเฟรมเวิร์ก

## เมื่อใช้ skill
ใช้ skill เมื่อโจทย์เกี่ยวกับหน้า list/detail, ฟอร์ม, loading/error/empty state, การเรียก API หรือแนวทางทดสอบฟรอนต์เอนด์

## ขั้นตอนเริ่มต้น
1. อ่าน `README.md`
2. เปิด `SKILLS_INDEX.md`
3. เลือก skill ที่ตรงกับงาน
4. เปิด prompt ที่ตรงกับงาน
5. ดูตัวอย่างประกอบ
