# บทเรียนการใช้งาน

## สถานการณ์
สร้างหน้าโปรไฟล์ผู้ใช้ที่มี loading, validation และ error recovery

## ขั้นตอนทีละขั้น
1. กำหนด page goal และ action หลัก
2. วาง state model: loading, ready, empty, error
3. เลือก skill ที่เกี่ยวข้อง
4. ใช้ prompt สำหรับวางแผนการทำงาน
5. เพิ่ม test สำหรับพฤติกรรมที่สำคัญ
