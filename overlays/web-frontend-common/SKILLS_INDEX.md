# Skills Index

Use these skills when the problem is frontend-agnostic.

## Core skills
- `auth-ux` — login, signup, recovery, and session UX
- `forms-validation-ux` — field validation, submit behavior, and feedback
- `loading-error-empty-states` — loading, error, empty, and retry states
- `api-consumption-patterns` — service boundaries, data flow, and caching shape
- `frontend-testing-basics` — behavior-oriented tests for frontend flows

## Selection guide
- New screen with several states -> `loading-error-empty-states`
- User input and submit behavior -> `forms-validation-ux`
- Login or account recovery UI -> `auth-ux`
- UI talking to an API -> `api-consumption-patterns`
- You need confidence from tests -> `frontend-testing-basics`
