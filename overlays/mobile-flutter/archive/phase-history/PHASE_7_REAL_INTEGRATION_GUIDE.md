# Phase 7: Real Integration Guide

This phase adds production-oriented integration guidance and starter code for:

- Firebase Auth wiring
- Google Maps adapter wiring
- Notification deep-link wiring with go_router
- Android / iOS / Web setup checklists
- Feature generator script

## Scope

This is still a starter kit, but it moves closer to a real repository baseline by including:
- environment and config templates
- route-intent bridge
- feature generation scaffolding
- platform setup notes
- stricter implementation prompts

## Remaining manual work
- Add actual package dependencies and versions
- Add Firebase project files
- Add Google Maps keys and native setup
- Add FCM native permissions and manifests
- Connect the placeholders to your backend/services
