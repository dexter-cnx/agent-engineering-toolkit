# Mobile Flutter Skill Index

This index is the catalog for all reusable Flutter skills in this overlay.

## Core

- [flutter-auth](./flutter-auth/README.md) — Authentication, session lifecycle, guarded routes, token refresh, logout hygiene.
- [flutter-permissions](./flutter-permissions/README.md) — Permission education, request orchestration, denied and permanently denied flows.
- [flutter-geolocation](./flutter-geolocation/README.md) — Current position, watch position, accuracy policy, service-disabled handling.
- [flutter-maps](./flutter-maps/README.md) — Map rendering, marker state, camera coordination, place picking, provider abstraction.
- [flutter-storage](./flutter-storage/README.md) — Secure storage, local cache, key namespaces, data retention, migration policy.
- [flutter-networking](./flutter-networking/README.md) — API clients, DTO mapping, retry and timeout policy, normalized errors.
- [flutter-deep-link](./flutter-deep-link/README.md) — Route schema, app links, universal links, redirect rules, fallback handling.
- [flutter-push-notifications](./flutter-push-notifications/README.md) — FCM/APNs orchestration, token registration, tap-to-route behavior.
- [flutter-i18n-l10n](./flutter-i18n-l10n/README.md) — Translation source of truth, key structure, fallback locales, formatting rules.
