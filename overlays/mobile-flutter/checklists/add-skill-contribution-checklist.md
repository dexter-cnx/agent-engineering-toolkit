# New Skill Contribution Checklist

- [ ] I checked whether an existing skill already covers the responsibility.
- [ ] I can explain why a new skill is needed.
- [ ] I used `tools/skillgen/bin/skillgen new` to scaffold the folder.
- [ ] The skill has a clear single responsibility.
- [ ] The skill has no placeholder text.
- [ ] The skill links at least one example and one template.
- [ ] The skill includes a validation checklist.
- [ ] The skill name matches the folder name.
- [ ] The skill does not duplicate an existing purpose or trigger set.
- [ ] I ran validation, overlap detection, docs sync, and index sync.
