# Add Notifications Feature

Use the mobile-flutter overlay to add production-ready notifications.

## Goals
- Support foreground/background/cold-start behavior
- Parse payloads safely
- Route notification taps through one navigation entrypoint

## Required skills
- flutter-dev
- flutter-notifications-fcm-production
- flutter-fcm-notifications
- flutter-deep-linking
- guide-clean-architecture-feature
- guide-app-release-checklist
- policy-clean-architecture
- policy-testing-minimum

## Deliverables
1. Notification architecture plan
2. Token lifecycle flow
3. Payload model and parser
4. Navigation handling
5. Test plan
6. Platform release checklist
