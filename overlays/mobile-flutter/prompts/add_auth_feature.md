# Add Firebase Auth Feature

Use the mobile-flutter overlay to add production-ready authentication.

## Goals
- Keep Firebase-specific code in the data layer
- Use Clean Architecture
- Add session bootstrap and route protection
- Support loading, error, unauthenticated, and authenticated states

- flutter-clean-architecture-audit
- flutter-firebase-auth-adapter
- flutter-firebase-auth-state
