# Web Frontend Checklist

- Confirm page and component boundaries stay separate.
- Confirm data fetching does not hide inside leaf UI components.
- Confirm shared design-system primitives remain reusable.
- Confirm state mutations stay in the correct layer.
- Confirm user-facing strings are localized where needed.
- Confirm responsive behavior is checked.
- Confirm accessibility basics are verified.
- Confirm tests exist for behavior changes.
- Confirm build and lint pass.
