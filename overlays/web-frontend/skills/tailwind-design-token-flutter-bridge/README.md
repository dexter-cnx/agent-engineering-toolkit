# Tailwind Design Token Flutter Bridge Skill

This skill helps agent teams keep web frontend and Flutter app styling aligned through one semantic token contract.

## What this skill covers

- Tailwind CSS usage rules for web frontend
- semantic design token rules
- Flutter mapping rules
- cross-platform naming contract
- prompts, templates, examples, and checklists

## Recommended usage

1. Read this skill before creating a new shared design system.
2. Decide semantic tokens before building many components.
3. Map tokens into Tailwind and Flutter separately.
4. Verify that both platforms still speak the same design language.
