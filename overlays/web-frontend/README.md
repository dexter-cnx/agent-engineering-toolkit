# Web Frontend Overlay

Use this overlay when the consuming repository is a web UI, dashboard, or product surface.

## Adds on top of foundation
- page/data/UI separation
- component hierarchy guidance
- design-system consistency expectations
- project-specific frontend verification hooks

## Recommended local additions in the consuming repo
- chosen framework conventions
- routing and state rules
- lint, test, and build commands
- design token or UI library constraints

## Overlay rule
This overlay extends the foundation.
It should not redefine the foundation's identity.
