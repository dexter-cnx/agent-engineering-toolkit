# unity-testing Skill

Use this skill when a Unity repo needs test coverage, validation seams, or smoke checks.

## Rules

- Separate Edit Mode and Play Mode tests by intent.
- Add seams that make scenes, prefabs, and services testable.
- Validate assumptions around missing references and bootstrap behavior.
- Keep smoke tests lightweight enough to run before merge.
- Prefer deterministic checks over manual verification where possible.

## Deliverables

- test scope
- seam plan
- smoke-check list
- verification notes
