# unity-memory-management Skill

Use this skill when memory lifetime, leaks, or pooling behavior needs review.

## Rules

- Track ownership for loaded assets and pooled objects.
- Check event subscriptions and delegates for leak risk.
- Release native or unmanaged resources explicitly and predictably.
- Treat pooling as a lifecycle contract, not just a performance tweak.
- Make memory-sensitive decisions visible in validation notes.

## Deliverables

- lifetime plan
- leak-risk notes
- pooling rules
- verification checklist
