# unity-analytics-and-crash-reporting Skill

Use this skill when telemetry, crash logging, or reporting contracts need structure.

## Rules

- Keep telemetry boundaries explicit and minimal.
- Name events consistently and review naming before shipping.
- Separate crash reporting from business logic.
- Prefer privacy-safe defaults and opt-in behavior where appropriate.
- Record only the data needed to support the product or diagnose failures.

## Deliverables

- event naming plan
- privacy notes
- crash-reporting notes
- verification checklist
