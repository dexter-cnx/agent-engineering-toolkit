# unity-multiplayer-foundation Skill

Use this skill when networking foundations need to stay provider-neutral and future-friendly.

## Rules

- Define authority, ownership, and trust boundaries early.
- Keep provider-specific networking code behind adapters.
- Treat prediction, replication, and rollback awareness as design inputs.
- Avoid locking the architecture to one netcode library too early.
- Keep multiplayer assumptions visible in review notes.

## Deliverables

- authority model notes
- adapter boundaries
- prediction awareness notes
- verification checklist
