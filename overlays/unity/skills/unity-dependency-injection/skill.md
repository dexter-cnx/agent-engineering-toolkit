# unity-dependency-injection Skill

Use this skill when object wiring, lifetime, or test seams need dependency-injection rules.

## Rules

- Keep construction and wiring separate from behavior.
- Make installer or composition-root lifecycles explicit.
- Prefer interfaces where they improve testability and boundary clarity.
- Avoid hidden service location when explicit injection is feasible.
- Keep container usage consistent across runtime and tests.

## Deliverables

- wiring plan
- lifetime notes
- testability notes
- verification checklist
