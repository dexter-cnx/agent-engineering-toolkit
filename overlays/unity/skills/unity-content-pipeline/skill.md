# unity-content-pipeline Skill

Use this skill when asset import or content organization needs a repeatable Unity pipeline.

## Rules

- Keep import settings policy-driven instead of ad hoc.
- Standardize naming and folder rules for assets and meta files.
- Make texture, audio, and model presets explicit and reviewable.
- Treat content validation as part of the build contract.
- Avoid hidden asset transformations that are hard to reproduce.

## Deliverables

- import policy
- naming rules
- meta-file discipline notes
- validation checklist
