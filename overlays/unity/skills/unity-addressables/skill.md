# unity-addressables Skill

Use this skill when asset loading or content delivery needs Addressables-specific structure.

## Rules

- Organize content by ownership, not by convenience.
- Keep labels and groups stable enough to support release and memory planning.
- Treat local versus remote content as an explicit product decision.
- Track load, release, and cache behavior in reviews.
- Keep memory lifecycle visible when assets are loaded asynchronously.

## Deliverables

- group and label plan
- content lifecycle notes
- release risk notes
- verification checklist
