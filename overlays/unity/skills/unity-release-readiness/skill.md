# unity-release-readiness Skill

Use this skill when a Unity release needs a final readiness pass.

## Rules

- Check scene inclusion, missing references, and asset duplication before release.
- Verify build output, target settings, and platform-specific packaging.
- Audit content and dependency risk as part of the release gate.
- Treat last-minute fixes as exceptions, not a substitute for validation.
- Record any known release risk explicitly.

## Deliverables

- release checklist
- asset audit notes
- build verification notes
- remaining risk log
