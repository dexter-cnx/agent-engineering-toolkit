# unity-ui-toolkit Skill

Use this skill when the project uses UI Toolkit and needs UI authoring or maintenance rules.

## Rules

- Keep UXML and USS organized by view or feature ownership.
- Use binding and presenters to keep view code thin.
- Treat panel settings and runtime/editor usage as explicit decisions.
- Avoid mixing Toolkit view logic with gameplay orchestration.
- Keep styling and structure changes easy to review in source control.

## Deliverables

- view structure plan
- binding strategy
- style ownership notes
- validation checklist
