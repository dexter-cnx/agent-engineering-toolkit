# unity-performance-profiling Skill

Use this skill when a Unity change needs performance investigation or a profiling review.

## Rules

- Start with profiler evidence, not guesses.
- Separate CPU, GPU, memory, and frame-time concerns.
- Watch for avoidable allocations and excessive object churn.
- Review draw calls, batching, and update frequency together.
- Confirm any optimization is measurable before it becomes a convention.

## Deliverables

- profiling plan
- bottleneck evidence
- optimization risk notes
- verification checklist
