# unity-state-machine Skill

Use this skill when state flow needs clarity across gameplay, UI, or animation.

## Rules

- Keep states small, named, and easy to test.
- Separate state transitions from side effects when possible.
- Integrate animation state deliberately instead of by accident.
- Avoid giant switch blocks that hide transition logic.
- Review entry, exit, and fallback behavior for each state.

## Deliverables

- state chart
- transition notes
- animation integration notes
- anti-pattern risks
