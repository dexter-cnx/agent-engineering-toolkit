# unity-editor-tooling Skill

Use this skill when editor automation, inspectors, or validation tooling needs a Unity convention.

## Rules

- Keep editor tooling focused on authoring and validation, not gameplay logic.
- Make validation commands easy to discover and rerun.
- Prefer explicit menu items or inspector controls for repeatable workflows.
- Keep automation safe enough for non-expert maintainers to use.
- Separate editor-only assemblies and dependencies from runtime code.

## Deliverables

- tooling scope
- menu or inspector plan
- validation behavior
- maintenance notes
