# วิธีใช้งานโอเวอร์เลย์นี้

## มันคืออะไร
โอเวอร์เลย์นี้ให้แนวคิด backend ที่คงรูปได้ข้าม runtime

## เมื่อใช้ skill
ใช้ skill เมื่อโจทย์เกี่ยวกับ contract, validation, auth shape, permission, file safety, email flow หรือการทดสอบ backend

## ขั้นตอนเริ่มต้น
1. อ่าน `README.md`
2. เปิด `SKILLS_INDEX.md`
3. ระบุปัญหา backend ให้ชัด
4. เลือก skill หนึ่งตัว
5. ใช้ prompt ที่ตรงกับงาน
