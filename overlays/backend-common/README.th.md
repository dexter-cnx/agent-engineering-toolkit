# โอเวอร์เลย์ Backend แบบทั่วไป

ใช้อโอเวอร์เลย์นี้สำหรับงาน backend ที่ควรเป็นกลางต่อ runtime: API contract, validation, auth, role/permission, audit logging, file safety, email flows และ backend testing strategy

## ใช้ร่วมกับ
- `overlays/backend-node/` สำหรับแนวทางเฉพาะ Node
- `overlays/python-service/` สำหรับแนวทางเฉพาะ Python service
- `overlays/backend-dotnet/` สำหรับรายละเอียด ASP.NET Core
- `docs/compositions/README.md` สำหรับตัวเลือกการประกอบ full-stack
- `docs/compositions/nextjs-dotnet/`
- `docs/compositions/nextjs-python-service/`
- `docs/compositions/nextjs-nodebackend/`
- `docs/backend-node-reuse-analysis.md` สำหรับเส้นทาง reuse ฝั่ง Node
