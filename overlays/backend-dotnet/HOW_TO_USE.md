# How to Use This Overlay

## What it is
This overlay gives you the .NET implementation layer for backend work.

## Minimal onboarding
1. Read `overlays/backend-common/README.md`
2. Read this overlay README
3. Choose the .NET-specific skill
4. Use the matching prompt file
5. Check the example for the expected shape

## Common tasks
| Task | Skill | Prompt |
| --- | --- | --- |
| Start a new ASP.NET Core project | `aspnet-project-structure` | `prompts/new_project.md` |
| Split layers and DI | `dotnet-di-layering` | `prompts/review_architecture.md` |
| Add EF Core access | `efcore-basics` | `prompts/generate_dotnet_endpoint.md` |
| Add JWT auth | `jwt-auth-dotnet` | `prompts/new_feature.md` |
| Add refresh tokens | `refresh-token-dotnet` | `prompts/new_feature.md` |
| Add validation | `fluentvalidation-dotnet` | `prompts/refactor_feature.md` |
