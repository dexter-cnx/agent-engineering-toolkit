# Tutorial

## Scenario
Build a secured ASP.NET Core endpoint for updating a profile with EF Core, validation, and health checks.

## Step-by-step
1. Use `aspnet-project-structure` to place the project files correctly.
2. Use `dotnet-di-layering` to keep services and repositories separated.
3. Use `efcore-basics` for persistence shape.
4. Use `jwt-auth-dotnet` and `refresh-token-dotnet` for auth flow.
5. Use `fluentvalidation-dotnet` for input validation.
6. Use `middleware-pipeline-dotnet` and `swagger-healthchecks-dotnet` for runtime polish.
