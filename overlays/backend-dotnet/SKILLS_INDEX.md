## Selection guide
- You are starting a .NET backend -> `aspnet-project-structure`
- You need layering decisions -> `dotnet-di-layering`
- You need persistence shape -> `efcore-basics`
- You need auth tokens -> `jwt-auth-dotnet`
- You need refresh lifecycle -> `refresh-token-dotnet`
- You need input validation -> `fluentvalidation-dotnet`
- You need middleware order -> `middleware-pipeline-dotnet`
- You need observability endpoints -> `swagger-healthchecks-dotnet`
