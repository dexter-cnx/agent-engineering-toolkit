# บทเรียนการใช้งาน

## สถานการณ์
สร้าง endpoint สำหรับอัปเดต profile ด้วย EF Core, validation และ health checks

## ขั้นตอนทีละขั้น
1. วางโครงสร้างโปรเจกต์ด้วย `aspnet-project-structure`
2. แยก layer และ DI ด้วย `dotnet-di-layering`
3. เพิ่ม persistence ด้วย `efcore-basics`
4. เพิ่ม auth ด้วย `jwt-auth-dotnet` และ `refresh-token-dotnet`
5. เพิ่ม validation และ runtime polish
