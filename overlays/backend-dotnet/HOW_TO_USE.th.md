# วิธีใช้งานโอเวอร์เลย์นี้

## มันคืออะไร
โอเวอร์เลย์นี้ให้ชั้น implementation ของ .NET สำหรับงาน backend

## เมื่อใช้ skill
ใช้ skill เมื่อโจทย์ต้องอาศัย ASP.NET Core structure, DI layering, EF Core, JWT, refresh token, validation หรือ middleware

## ขั้นตอนเริ่มต้น
1. อ่าน `overlays/backend-common/README.md`
2. อ่าน `README.md` ของโอเวอร์เลย์นี้
3. เลือก skill เฉพาะ .NET
4. ใช้ prompt ที่ตรงกับงาน
5. ดูตัวอย่างเพื่ออ้างอิงรูปแบบ
