# โอเวอร์เลย์ Backend .NET

ใช้อโอเวอร์เลย์นี้เมื่อ backend ใช้ ASP.NET Core หรือ .NET และต้องการแนวทางเรื่อง project structure, DI layering, EF Core, JWT/refresh token, FluentValidation, middleware และ health checks

## ใช้ร่วมกับ
- `overlays/backend-common/` สำหรับแนวคิด backend แบบกลาง runtime
- `docs/compositions/README.md` สำหรับตัวเลือกการประกอบ full-stack
- `docs/compositions/nextjs-dotnet/` สำหรับ Next.js + ASP.NET Core
