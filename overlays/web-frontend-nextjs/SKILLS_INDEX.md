# Skills Index

## Core skills
- `app-router-structure` — page, route group, and layout placement
- `server-client-boundaries` — choose server versus client ownership
- `route-handlers` — thin HTTP entry points
- `middleware-protected-routes` — protection and redirects
- `nextjs-data-fetching` — read path, caching, and invalidation
- `nextjs-auth-integration` — auth session consumption in the frontend

## Selection guide
- You need to place a page correctly -> `app-router-structure`
- You need to split logic between server and client -> `server-client-boundaries`
- You need an HTTP entry point -> `route-handlers`
- You need route protection -> `middleware-protected-routes`
- You need data loading behavior -> `nextjs-data-fetching`
- You need auth-aware frontend behavior -> `nextjs-auth-integration`
