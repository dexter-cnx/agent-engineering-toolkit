# โอเวอร์เลย์เว็บฟรอนต์เอนด์ Next.js

ใช้อโอเวอร์เลย์นี้เมื่อ frontend ใช้ Next.js และต้องการแนวทางเรื่อง App Router, server/client boundary, route handler, middleware, data fetching และ auth integration

## ใช้ร่วมกับ
- `overlays/web-frontend-common/` สำหรับ pattern ฟรอนต์เอนด์ที่ใช้ซ้ำได้
- `docs/compositions/README.md` สำหรับตัวเลือกการประกอบ full-stack
- `docs/compositions/nextjs-dotnet/` สำหรับ Next.js + ASP.NET Core
- `docs/compositions/nextjs-python-service/` สำหรับ Next.js + Python service
- `docs/compositions/nextjs-nodebackend/` สำหรับ Next.js + Node backend
