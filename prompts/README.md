# Prompt Runtime (Canonical)

Canonical model: **source prompts → provider pack manifests → compiled runtime prompts**.

## Canonical source prompts

- `prompts/system/operating-contract.md`
- `prompts/tasks/repo-hardening.md`
- `prompts/tasks/release-readiness.md`

These files are the editable source of truth.

## Provider packs

- `prompts/packs/codex/pack-manifest.md`
- `prompts/packs/claude/pack-manifest.md`
- `prompts/packs/gemini/pack-manifest.md`

Packs specify which source prompts compile into each provider runtime.

## Compiled prompts

- `prompts/compiled/codex-runtime.md`
- `prompts/compiled/claude-runtime.md`
- `prompts/compiled/gemini-runtime.md`

Compiled prompts are generated by `python3 tools/prompts/compile_prompts.py`.
Do not manually edit compiled outputs.

## Boundary note

`INDEX_PROMPTS.md` is a legacy/frozen root index and not canonical.
Use this README and `docs/reference/prompt-catalog.md`.
