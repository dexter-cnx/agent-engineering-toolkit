# Mobile Flutter 23-Skill Pack

This pack is structured so you can extract it at the repository root of `agent-engineering-toolkit`.

## What is included
- updated root documentation files
- updated overlay documentation for `overlays/mobile-flutter`
- a complete mobile Flutter skill catalog with 23 reusable skills
- examples, prompts, checklists, and templates for each skill
- a validation report generated from this pack

## Suggested apply flow
1. Extract this archive at the repository root.
2. Review the changed files.
3. Run your repository checks.
4. Commit with a docs-and-overlay focused message.

## Notes
- This pack does not contain secrets or environment-specific credentials.
- All internal markdown references in this pack were validated locally before packaging.
- The pack is designed to extend the existing foundation identity rather than replace it.
